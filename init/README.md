### Робота з частинами коду 
- Код можна вставляти як текст(так робити не правильно)


    def greet(name):
       print (f"HELLO,{name}!")

Вставляємо код через блоки з підсвіткою синтаксису (так правильно)

```python
    def greet(name):
        print (f"HELLO,{name}!")
```
